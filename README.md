# ADAPT-AI Community Platform

## Project summary

ADAPT-AI is a web-based learning platform designed to help K-12 teachers create personalized, AI-generated classroom materials using an evidence-centered design framework called NGSS. Our capstone contribution focuses on building the platform's online community module, which allows educators to share, discover, and reuse teaching resources with one another. Through features like post creation, commenting, saving, and filtering, the community module transforms ADAPT-AI from a solo content generation tool into a collaborative ecosystem where teachers can learn from each other and build on each other's work.
