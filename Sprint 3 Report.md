# Sprint 3 Report

## What's New
* New Questions panel with List UI
* New UI for existing Tasks panel
* New template option for creating tasks

## Work Summary
During Sprint 3, the team made significant strides in refining the Adapt-AI platform's core functionality. The Community tab was restructured into two distinct panels — Questions and Tasks — topped by a new hero header block. The Tasks panel retained its existing functionality but received a complete UI overhaul, while the Questions panel was introduced as a brand-new feature per client request, offering a GitHub-style list interface that allows instructors to post questions related to tasks and the platform. Additionally, the sprint delivered a new template option to streamline task creation and added the ability for tasks to be published, further expanding the platform's instructional workflow capabilities.
